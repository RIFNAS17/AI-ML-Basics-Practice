# AI-ML Basics Practice

This repository contains practice notebooks for learning fundamental AI/ML concepts.

## 📂 Contents
1. **AI_ML_Terms.ipynb** – 15 glossary notes explaining key AI/ML terms (Overfitting, Precision, Recall, etc.).
2. **Visualizations.ipynb** – Charts and graphs using Iris and Titanic datasets.
3. **Datasets_Exploration.ipynb** – Exploration of 5 datasets (Iris, Titanic, Wine Quality, MNIST, Heart Disease).
4. **Learning_Summary.ipynb** – Short notes summarizing supervised, unsupervised, and reinforcement learning.

## 🎯 Purpose
- Build strong AI/ML fundamentals.  
- Practice Python libraries (Pandas, Matplotlib, Seaborn, Scikit-learn).  
- Create recruiter-friendly projects for resume and placements.  

## 🛠️ Libraries Used
- Python 3  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## 🚀 How to Use
- Open each notebook in Jupyter Notebook or Google Colab.  
- Run the cells to see definitions, visualizations, and dataset analysis.  
- Read the Markdown notes for learning summaries.
